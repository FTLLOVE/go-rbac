# Golang使用RBAC模式来实现权限到按钮级别的权限管理
> 权限系统基本上在我们使用的系统中都很常见，而权限的要求也是各不一样，有的仅需要精确到菜单，
> 而有的颗粒度需要精确到具体菜单的某个按钮或者方法，所以这样就带来了系统设计的复杂度，由此
> 我使用golang不借助任何框架从0来实现一个颗粒度相对来说比较高的系统.

### 相关技术
```
go version: 1.15.1
go frame:  v1.15.6
mysql: v8
```
